# GameHub - Ultimate Gaming Portal

GameHub is a modern gaming platform where users can explore game details, manage their profiles, and stay connected with the gaming community. This is a React-based web application secured with Firebase Authentication.

**Live URL:**  
https://animated-parfait-d0aae7.netlify.app/profile

---

## Purpose

The main goal of this project is to provide a seamless gaming information portal for users. Key objectives include:

- Browsing detailed information about various games.
- Providing a secure login and registration system.
- Allowing users to recover/reset passwords if forgotten.
- Enabling users to customize and update their personal profiles.

---

## Key Features

- **Firebase Authentication:** Secure login via Email/Password and Google.
- **Dynamic Routing:** Smooth page transitions using React Router v7.
- **Protected Routes:** Restricts access to private pages (like Profile or Game Details) for unauthenticated users.
- **Password Reset:** Functional password recovery system via email.
- **Responsive UI:** Fully optimized for Mobile, Tablet, and Desktop devices using Tailwind CSS.
- **Profile Customization:** Users can update their display names and profile picture URLs.
